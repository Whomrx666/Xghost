# Xghost
![Xghost preview](Xghost.jpg)

## introduction
Xghost is a pentesting tool designed to independently search for vulnerabilities that systematizes what attacks can be carried out.

## Instalations
```
$ pkg update -y && pkg upgrade -y
$ pkg install git -y
$ git clone https://github.com/Whomrx666/Xghost.git 
$ cd Xghost
$ python3 install.py

```
## Run manually
```
$ python3 Xghost.py
```

## Instructions
- **First**: Install the tools according to the instructions above.
- **Second**: Run the main tools after installation.
- **Third**: Just enter the target domain directly into the tool.
- **Last**: Automatic tools perform scanning and look for security vulnerabilities and suitable attacks to carry out.
